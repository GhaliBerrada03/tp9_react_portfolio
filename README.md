# Mon Portfolio

Portfolio personnel développé avec React, TypeScript et Tailwind CSS v4.

## Stack

- React 19 + TypeScript
- Vite 8
- Tailwind CSS v4
- Framer Motion
- React Router v7

## Lancer le projet
```bash
npm install
npm run dev
```

## Structure
```
src/
├── app/        # Layout et routing
├── components/ # Composants réutilisables
├── data/       # Contenu (projets, certifs, profil)
└── pages/      # Pages du site
```
